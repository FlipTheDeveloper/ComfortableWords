What makes a word hard to type?

There are some words that are easier to type than others. Some words are awful to type, like "lollipop". But other words such as "Gabriel" are really fun to type.

My theory is that if you have to move one finger to type subsequent keys in a word, the rest of your fingers need to wait for that one finger to move and press that key.

Since typing the same key twice can be kind of cumbersome, the application rates it as being half as hard as other shared jurisdiction key presses.

To display this theory, I wrote this javascript application to display how many times there are subsequent keys within a finger's "jurisdiction" on the keyboard. When you type in a sentence or word, and it finds keys typed that share the jurisdiction of the previously typed key. And rates the word on how hard it is to type.
